       <li><strong>1973</strong>-Chairman of the Presidential Advisory<sub>Committee</sub> for the Peace Corps.</li>
       <li><strong>1982</strong> -Armstrong was chairman of Computing Technologies for Aviation, Inc., Charlottesville, Va. until 1992. </li>
